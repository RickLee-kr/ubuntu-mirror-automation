Ubuntu Mirror Server - Setup Guide
Ubuntu Mirror Server - Complete Setup Guide
Purpose: Step-by-step instructions to set up an Ubuntu Package Mirror Server for
offline OS upgrades  
Target Versions: Ubuntu 16.04 → 18.04 → 20.04 → 22.04 → 24.04  
Time Required: 1-2 hours setup + 12-24 hours initial sync  
Last Updated: January 31, 2026
{panel:bgColor=#E3FCEF|borderColor=#00875A}
✅  Prerequisites Checklist
Before starting, ensure you have:
☑  Ubuntu 24.04 LTS server installed
☑  Two disks configured:
OS Disk: 100 GB
Data Disk: 1 TB mounted at /var/spool/apt-mirror
☑  Root or sudo access
☑  Internet connectivity (100+ Mbps recommended)
☑  Static IP address configured
